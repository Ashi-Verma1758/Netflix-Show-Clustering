Netflix Show Clustering and Recommendation System

Overview
This project uses Machine Learning and Natural Language Processing
to cluster Netflix content and recommend similar shows.

Features
- Data Cleaning
- Exploratory Data Analysis
- TF-IDF Vectorization
- K-Means Clustering
- Cluster Visualization
- Recommendation Engine
- Streamlit Dashboard

Tech Stack
- Python
- Pandas
- Scikit-Learn
- Streamlit
- NLP
